JAVA의 리스트
    1. 선언
        ArrayList<타입> 리스트변수명 = new ArrayList<>();
        ex) 문자열 리스트 : ArrayList<String> 변수명 = new ArrayList<>();
        ex) 정수 리스트 : ArrayList<Integer>
        ex) 실수 리스트 : ArrayList<Double>
            int -> Integer : 기본 자료형을 클래스처럼 사용할 때 사용되는 Wrapper 클래스
                - 기본 자료형은 리터럴만 다루기 때문에 추가적인 기능/속성을 다루기 위해 클래스화
        e.g. 사용자 정의 타입 리스트 : ArrayList<클래스명>

    2. 리스트 내 요소 추가
        리스트변수명.add(객체);   : 해당 객체를 리스트에 추가

    3. 리스트 내 요소 호출
        리스트변수명.get(인덱스);  : 해당 인덱스의 요소(객체) 반환

    4. 리스트 내 요소 삭제
        리스트변수명.remove(인덱스); : 해당 인덱스의 요소(객체) 삭제 및 뒷부분 한칸씩 앞당김

    5. 리스트 내 요소 총 갯수
        리스트변수명.size();

    6. 리스트 내 요소 찾기
        리스트변수명.indexOf(값 또는 객체); : 리스트 내 값 또는 객체가 존재하면 찾은 인덱스, 없으면 -1 반환 (인덱스 못 찾음)
        리스트변수명.contains();           : 논리값 반환 true/false

    7. 리스트 내 비어있는지 확인
        리스트변수명.isEmpty();

    8. 리스트 순회 (리스트 처음 요소/인덱스부터 마지막 요소/인덱스까지 호출)
        1. for ( int i = 0 ; i < 리스트변수명.size() ; i++){ 명령 }
        2. for (타입 반복변수명 : 리스트변수명) { 명령 }
        3. 리스트변수명.forEach ( 반복변수명 -> 명령 (sout 등) );
