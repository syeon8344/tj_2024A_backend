
[컴퓨터]
    [메모리 종류]
        - 주기억장치         : 메모리 ( RAM )
            - 실행중인 메모리 관리, 휘발성, 프로그램이 꺼지거나 전원이 꺼지면 사라진다
            - JVM
            - 실행중인 프로그램 내 메모리만 저장하기 때문에 영구저장이 불가능하다.

        - 보조기억장치       : SSD, HDD ( ROM )
            - 비실행중인 메모리 관리, 비휘발성, 전원이 꺼져도 저장이 가능
            - 파일(메모장) 처리, 데이터베이스
            - C 드라이브, USB 드라이브 등 : 직접 삭제하기 전까지 영구저장

자바는 JVM (메모리 관리) 라는 구조로 메모리 관리, 휘발성

데이터베이스 : 액셀, 액세스, 한글 등등 데이터베이스 기반 프로그램
데이터베이스
    1. 데이터 : 순수한 자료
    2. 베이스 : 모임/집합
    ---> 자료 집합

데이터베이스 종류
    1. 트리형태 2. 네트워크형 3. 관계형 4. NoSQL
    학습&실무 : 관계형 데이터베이스

관계형 데이터베이스 :
    1. 행과 열로 이루어진 데이터의 집합 ==> 표/테이블
    2. 표/테이블을 고유/식별 값을 참조하여 서로 다른 테이블끼리 종속되는 관계

관계형 데이터베이스 사용하는 프로그램
    1. Oracle 2. MySQL 3. MariaDB...

알아야 하는 용어
    DB : 데이터베이스 ( = 데이터의 집합 => 표)
    DBA : 데이터베이스 관리자
    DBMS : 데이터베이스 관리 프로그램 ( Oracle, MySQL, MariaDB...)
    DB SERVER : 데이터베이스가 설치된 PC
        - SERVER : 자원을 제공
        - CLIENT : 자원을 요청
    SQL : 데이터베이스 시스템 프로그램이 이해할 수 있는 언어 --> 문법
        예]
        가계부 표 만들어줘 ---> create table 가계부{}
